# Banking API Project

This project is a simple and secure banking API built with Flask. It allows you to manage user accounts and handle transactions.

## Documentation

Read the API documentation here: https://documenter.getpostman.com/view/39359413/2sAY4vh3Yt

## Deployment

The API is deployed and can be accessed here: http://other-rubetta-cicely-12f8a027.koyeb.app/

## How to Run the Project

1. **Clone the Repository**:
   Type this in the terminal:
   - git clone https://github.com/revou-fsse-5/milestone-4-ciceiy.git
   - cd milestone-4-ciceiy
2. **Install Dependencies**:
   Make sure you have Python and pip installed. Then, install the required packages by typing this in the terminal:<br>
   pip install -r requirements.txt
3. **Set Up the Database**:
   Make sure you have a MySQL database set up.
4. **Run the Aplication**:
   Type this in the terminal:<br>
   flask run
